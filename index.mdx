HealthMambo is an open-source clinical management system built specifically for physiotherapy clinics operating in resource-limited environments. It unifies appointment scheduling, staff management, and a credit-based payment system into a single self-hosted web platform — with no licensing fees.

## What HealthMambo does

HealthMambo solves three core operational challenges for physiotherapy clinics:

- **Appointment management** — A guided multi-step booking wizard prevents double-booking and assigns the right physiotherapist to each patient.
- **Credit-based billing** — Patients purchase credit packages that are automatically deducted when services are booked, simplifying payment tracking.
- **Role-based access** — Four distinct user roles (Admin, Receptionist, Physiotherapist, Patient) give each person exactly the access they need.

## Getting started

<Steps>
  <Step title="Install prerequisites">
    Install .NET 8 Runtime and SQL Server Express on a Windows machine.
  </Step>
  <Step title="Clone and configure">
    Clone the repository and update the database connection string in `appsettings.json`.
  </Step>
  <Step title="Run database migrations">
    Apply Entity Framework migrations to create the database schema.
  </Step>
  <Step title="Create your first admin user">
    Launch the app, register the first account, and assign it the Admin role through the database.
  </Step>
</Steps>

<Note>
  HealthMambo is designed to run on a local network server, minimizing internet dependency and making it suitable for clinics with limited connectivity.
</Note>
